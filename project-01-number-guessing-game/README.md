# 🎮 Number Guessing Game

## 🎯 Objective

Build a simple number guessing game using C programming.

---

## 📌 Description

This program generates a random number between **1 and 100**. The user keeps guessing until the correct number is found. After each guess, the program provides feedback to help the user reach the correct answer.

---

## 🧠 Concepts Used

- Random Number Generation (`rand()`, `srand()`)
- Time Library (`time()`)
- Variables
- Conditional Statements (`if`, `else if`, `else`)
- Loops (`do-while`)
- User Input (`scanf`)
- Output (`printf`)

---
